# ALX Listing App

A simple Airbnb clone listing page built with Next.js, TypeScript, and Tailwind CSS.
The project aims to provide a foundation for displaying property listings with reusable components, organized project structure, and responsive design.

## Project Structure

- `components/`: Reusable UI components like Card and Button.
- `interfaces/`: TypeScript interfaces for components and app data.
- `constants/`: Store reusable constants, API URLs, and configuration.
- `public/assets/`: All images, SVGs, and static assets used across the project.

## Getting Started

1. **Install dependencies**:

```bash
npm install
```
